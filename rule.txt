Test Entity Component System simple

Ce test se déroule sur Unity, étant le seul moteur qui a de vrai ressources sur la question.
La documentation contient toutes informations et liens nécessaires : https://docs.unity3d.com/Packages/com.unity.entities@1.3/manual/index.html

Etape 1
- Installer les packages dans un nouveau projet Unity
- Créer une voiture (un cube rectangulaire) avec la capacité de déplacement et de rotation sur l'axe Y.

Etape 2
- Créer un spawner de voiture configurable dans la scene (vitesse, couleur...)
- Ajouter un set de couleurs sur chaque spawner qui permettra "au spawner" de sélectionner une couleur aléatoire pour la voiture

Etape 3:
- Permettre de créer des routes avec plusieurs segments (angle droit ou angle plat)
- Les routes sont configurables dans la scene
- Chaque spawner a une route affectée et la voiture spawn doit la parcourir et disparaître à la fin
